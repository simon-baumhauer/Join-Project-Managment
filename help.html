Help

Hier finden Sie die benötigte Hilfe zu unserer Join Kanban:

  • ADDTASK:

    Hier können Sie Aufgaben erstellen und verfügbares Personal zuteilen.

  • BACKLOG:

    Hier können Sie alle Aktivitäten oder Aufgaben, die erstellt wurden, einsehen.

        Main Funktion

      • Wenn Sie mit dem Mauszeiger auf die Bilder von den Kollegen gehen, werden die Informationen von den jeweiligen Personen angezeigt
      • Die Details jeder Aufgabe können Sie jederzeit bearbeiten, in dem Sie auf den Text klicken.
      • Wenn Sie hier drauf klicken, werden die Tasks an Board weitergeleitet.

        • Farbe

      • Die Aufgaben mit hoher Priorität werden rot umrandet.
      • Die Aufgaben mit mittlerer Priorität werden gelb umrandet.
      • Die Aufgaben mit niedriger Priorität werden grün umrandet.

  • Board:

     Hier können Sie Ihre Aufgaben, nachdem sie von Backlog weitergeleitet wurden, ansehen.

        Main Funktion

      • Hier können Sie Ihre Aufgaben ganz einfach mit Hilfe der Maus von einer Liste z.B Todo zur anderen Liste z.B Progress per Drag und Drop verschieben.
      • Wenn Sie auf die jeweilige Aufgabe klicken, werden alle Informationen zur jeweiligen Aufgabe in einer Infobox eingebelendet und wenn Sie außerhalb der Infobox klicken, werden die Informationen wieder ausgeblendet.
      • Wenn Sie in der Infobox mit dem Mauszeiger auf die Bilder von den Kollegen gehen, werden die Informationen von den jeweiligen Personen (wie in Backlog) angezeigt.

        • Farbe

      • Die Aufgaben mit hoher Priorität werden rot hinterlegt.
      • Die Aufgaben mit mittlerer Priorität werden gelb hinterlegt.
      • Die Aufgaben mit niedriger Priorität werden grün hinterlegt.
